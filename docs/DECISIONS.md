# DECISIONS — 決定記録

## 採用済み

### D001 Excel/CSVから開始

初期はExcel/CSVで運用し、限界が来たら他システムへ移行する。

### D002 Lv0は最小3列

`ID,林道名,取得元`

### D003 県コードは3文字

福島/福岡等の混同を避けるため。

長野県は `NGN`。

### D004 情報信頼性を重視

情報源を必ず残す。

### D005 全林道を一件ずつ深掘りしない

まずLv0で全体像を作り、必要な林道だけ成長させる。

### D006 高速モード/完全モード

直近利用と長期品質を両立する。

### D007 今週末MVP優先

理想設計より、スマホ・共有・地図・ナビの一連動作を優先。

### D008 カルテはマスターから生成する表示用情報

事実と解釈を分離する。

### D009 背景地図は公式配信方式だけを使う

- OSM、国土地理院淡色地図、地理院航空写真はLeafletのタイルレイヤーとして切り替える
- OpenFreeMapは公式LibertyスタイルをMapLibre GL Leafletで表示する
- MapLibre本体とLeaflet連携ライブラリはアプリ内へ同梱し、起動時のCDN依存を作らない
- Googleマップは Maps JavaScript API を使い、非公式タイルURLは使わない
- Google APIキーが未設定・無効の場合はLeaflet地図を維持して案内を表示する
- 公開用Google APIキーにはHTTPリファラー制限を設定する
