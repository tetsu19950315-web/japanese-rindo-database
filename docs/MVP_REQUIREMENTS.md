# MVP_REQUIREMENTS — 今週末版

## 対象ユーザー

本人と林道アタック仲間。

## 対象地域

第一優先:

- 茅野市
- 諏訪市

必要に応じて:

- 原村
- 富士見町
- 岡谷市
- 下諏訪町

## 必須ユーザーフロー

1. 仲間が共有URLをスマホで開く
2. 地図で候補林道を確認する
3. 林道をタップする
4. 重要情報を見る
5. 「入口までナビ」を押す
6. Google Mapsで経路案内を開始する

## 必須機能

- モバイル対応
- 地図表示
- 候補林道表示
- 林道名表示
- カルテ表示
- ナビリンク
- URL共有

## あると良い

- 現在地
- 難易度や通行状況のアイコン
- 候補/実走済みフィルター
- 写真

## 非目標

- 独自ターンバイターンナビ
- 林道内自動ルーティング
- 大規模SNS
- 全国完全網羅
- 高度なユーザー管理

## 受け入れ条件

### Mobile

幅360px前後でも主要操作ができる。

### Map

茅野・諏訪周辺の候補が10件以上表示される。

### Karte

実走候補3〜5本で重要情報が見られる。

### Navigation

ボタン1回でGoogle Mapsの経路検索に遷移する。

### Sharing

仲間がアカウント作成なしでURLから閲覧できる。
