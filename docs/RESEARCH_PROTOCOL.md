# RESEARCH_PROTOCOL — Lv0マスター標準仕様 v1.3

## 目的

指定地域に存在する林道を一覧化する。

Lv0では次を調査しない。

- 未舗装判定
- 難易度
- 距離
- 景観
- 通行可否
- カルテ

## モード

### 高速モード

- 今後の育成前提
- 網羅性より速度
- 取得できた情報のみ登録
- 取得漏れ許容
- 実行レポートを残す

### 完全モード

- 最終版Lv0作成
- 行政資料等を可能な限り横断
- 網羅性最優先
- 取得漏れ最小化

## 情報源優先順位

1. 行政資料
2. OpenStreetMap
3. その他公開情報（GIS、国土地理院等）
4. ブログ・個人サイト（補完のみ）

## 基本ルール

- AI推測禁止
- 存在確認できない林道は登録しない
- 正式名称優先
- 重複禁止
- 情報源必須

## CSV

```csv
ID,林道名,取得元
```

## ID

`都道府県3文字コード-6桁連番`

長野県:

`NGN-000001`

## 実行レポート

必須:

- 実行モード
- 検索対象地域
- 取得件数
- 重複除去件数
- 最終登録件数
- 主な情報源
- 取得できなかった主な情報源
- 不足資料
- 網羅率自己評価（高速モードのみ）

## 重要

既存IDは保持する。

新しい林道は末尾に追加する。ソートのためにIDを振り直さない。
