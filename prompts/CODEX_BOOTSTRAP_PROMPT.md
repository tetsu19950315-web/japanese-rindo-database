# Codex開始プロンプト

このリポジトリを「日本林道データベース」プロジェクトとして引き継いでください。

最初に `AGENTS.md`、`STATUS.md`、`TASKS.md`、`PLANS.md`、`docs/MVP_REQUIREMENTS.md` を読んでください。

最優先目標は、2026年7月11〜12日の長野県での林道アタックに間に合う、仲間内で使えるMVPを完成させることです。

MVP要件は以下です。

- スマホで動く
- URLで仲間と共有できる
- 茅野・諏訪周辺の林道候補を地図で視覚的に見られる
- 林道をタップすると重要情報（カルテ）が見られる
- ボタン1回でGoogle Mapsを開き、林道入口までナビを開始できる

作業は `TASKS.md` のP0を上から順に進めてください。

重要:

- 追加の理想設計で作業を止めないでください
- 今週末MVPに不要な機能は実装しないでください
- 判断できることは自律的に決めて進めてください
- 確認が必要なのは、費用発生、認証情報、破壊的変更、公開範囲など重大な判断だけです
- 林道データを創作しないでください
- 既存IDを振り直さないでください
- 情報源を必ず残してください
- 長い作業では `PLANS.md` を更新し、完了時に `STATUS.md` と `TASKS.md` を更新してください

まず次を実行してください。

1. 既存の長野県Lv0 CSVを検証する
2. 茅野・諏訪周辺の候補林道を10〜30件抽出する
3. 地図表示に必要な位置情報を10件以上集めるための最短手順を実行する
4. 静的Web地図MVPを作り始める
5. 途中成果でもローカルで動く状態を保つ

最初の返答では、長い説明ではなく、

- 読み取った現在地
- これから実行するP0タスク
- すぐ着手する最初の変更

を簡潔に示し、そのまま作業を開始してください。
